## SI 1: Description of Beaver Foraging Index (BFI) values 
<br/>

| BVI Value       | Definition           |
| :-------------: |:--------------------:|
| 0               | No Vegetation        |
| 1               | Unsuitable           |
| 2               | Barely Suitable      |
| 3               | Moderately Suitable  |
| 4               | Suitable             |
| 5               | Preferred            |
